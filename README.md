# Trabajo_IAS
Trabajo de curso de la asignatura Instrumentación y Acondicionamiento de señal

En este repositorio se encuentran los archivos de LTSpice correspondientes al trabajo de curso de Instrumentación y Acondicionamiento de Señal de 4º GIERM.

## Carpeta Definitivo

En esta carpeta se encuentran las versiones finales de cada topología. Cada simulación está configurada para probar el funcionamiento correcto de cada circuito completo; es decir, con todas sus funcionalidades activadas (medida, integración, reset y select).

## Carpeta Versiones anteriores

Aquí se encuentran las etapas de implementación y comprobación paso a paso de cada topología. Cada simulación viene con medidas y parámetros de prueba para comprobar el funcionamiento de lo que se desea conseguir en esa etapa.

## Carpeta Comparacion

Contiene las versiones finales de cada circuito, pero con condiciones de simulación normalizadas y medidas para distintas métricas de rendimiento que se usan para comparar las topologías entre ellas.
